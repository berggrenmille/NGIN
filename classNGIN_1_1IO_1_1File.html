Detailed Description

template<typename StoragePolicy = TypeErasure::DynamicStoragePolicy>
class NGIN::IO::File< StoragePolicy >

A class representing a generic File object.

Template Parameters
StoragePolicyThe storage policy to use, defaults to NGIN::TypeErasure::DynamicStoragePolicy.
