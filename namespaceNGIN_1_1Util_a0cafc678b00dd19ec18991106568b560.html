std::string NGIN::Util::RuntimeFormat ( std::string_view  message,

)

Formats a string at runtime using provided format arguments. This function wraps around the fmt::format function from the fmt library.

Note
For compile-time string formatting, see NGIN::Util::Format function.
Template Parameters
ArgsTypes of the arguments used for formatting.
Parameters
messageThe message string to be formatted.
argsThe arguments to be substituted into the format string.
Returns
The formatted string.
std::string_view msg = "Hello, {}!";
std::string result = RuntimeFormat(msg, "World");
// result will be "Hello, World!"
