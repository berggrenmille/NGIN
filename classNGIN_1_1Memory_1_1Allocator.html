Detailed Description

template<typename StoragePolicy>
class NGIN::Memory::Allocator< StoragePolicy >

Represents a generic memory allocator with dynamic dispatch.

Template Parameters
StoragePolicyThe storage policy for wrapped allocator.

The Allocator class manages objects with the expected Allocate, Deallocate, and DeallocateAll interface. Instead of using virtual functions, it employs manual vtable pointers for dynamic dispatch.
