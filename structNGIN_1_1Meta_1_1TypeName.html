Detailed Description

template<typename T>
struct NGIN::Meta::TypeName< T >

Retrieves the type name as a string.

This struct provides functionality to get the type name of a given type as a compile-time StringView.

Template Parameters
TThe type whose name is to be retrieved.
namsepace Foo
{
struct Bar{};
}
std::cout << TypeName<Foo::Bar>::Class(); // Outputs: Bar
std::cout << TypeName<Foo::Bar>::Namespace(); // Outputs: Foo
std::cout << TypeName<Foo::Bar>::Full(); // Outputs: Foo::Bar
